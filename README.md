# NIS3356 Homework

信息内容安全课程设计小组仓库。

## 目录

- `ASR/`：语音识别与安全审计模块，包含远程已有 ASR 工作。
- `NLP/`：文本处理与主题分类模块。
- `图像处理/`：基于 PaddleOCR 的社交媒体热点图像 OCR 与结构化输出模块。
- `数据爬取/`：微博热搜 Top50 快照、代表帖子文本与多媒体爬取模块。
- `数据分析/`：热点趋势统计与预测分析模块。

当前已同步：

- `ASR/`：ASR 安全审计系统。
- `图像处理/`：图像 OCR、预处理、多版本择优、真实截图采集、LLM 精简输出。
- `数据爬取/`：微博热搜条目固定快照、候选帖子抓取、代表帖筛选、图片/音视频链接与文件输出。
