# NLP 主题发现模块说明

## 模块位置
- 核心代码：`topic_discovery.py`
- 依赖：`preprocessing.py`

## 概述
实现两种经典主题发现方法：

| 方法 | 原理 | 特点 |
|------|------|------|
| **LDA** | Collapsed Gibbs Sampling | 概率主题模型，文档=主题的混合 |
| **K-means** | 聚类 + 簇内关键词提取 | 简单快速，硬聚类 |

## 算法原理

### LDA (Gibbs Sampling)
1. 对每个词随机分配主题
2. 迭代重采样每个词的主题：P(z|w) ∝ (n_d+α) × (n_w+β) / (Σn_w+Vβ)
3. 收敛后计算：φ(主题-词分布)、θ(文档-主题分布)
4. 参数：α(文档主题先验)、β(主题词先验)

### K-means 聚类
1. 词频向量化
2. 随机初始化聚类中心
3. 迭代分配+更新质心（Cosine距离）
4. 簇内TF-IDF提取关键词

## 快速使用
```python
from topic_discovery import TopicDiscoverer

# LDA
td = TopicDiscoverer(method="lda", n_topics=8)
td.fit(texts)
topics = td.get_topics(top_n=10)
# [{"topic_id": 0, "words": ["词1","词2",...], "weight": 0.15}, ...]

# K-means
km = TopicDiscoverer(method="kmeans", n_topics=8)
km.fit(texts)
topics2 = km.get_topics(top_n=10)

# 新文档主题推断（仅LDA）
theta = td.transform(["新文本"])
```

## 参数说明
| 参数 | 默认 | 方法 | 说明 |
|------|------|------|------|
| n_topics | 10 | 通用 | 主题/簇数量 |
| alpha | 0.1 | LDA | 文档-主题先验 |
| beta | 0.01 | LDA | 主题-词先验 |
| n_iter | 200 | LDA | Gibbs采样迭代次数 |
