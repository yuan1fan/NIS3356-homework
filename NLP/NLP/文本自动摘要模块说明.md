# NLP 自动摘要模块说明

## 模块位置
- 核心代码：`text_summarization.py`
- 依赖：`preprocessing.py`

## 概述
实现两种经典抽取式摘要方法：

| 方法 | 原理 | 特点 |
|------|------|------|
| **TextRank** | 句子相似度图 + PageRank | 图排序，无需外部语料 |
| **TF-IDF** | 关键词密度 + 位置奖励 | 简单快速，偏向信息密度 |

## 算法原理

### TextRank 句子摘要
1. 将文本分割为句子
2. 对每个句子分词，构建 TF 向量（L2 归一化）
3. 构建句子相似度图（Cosine 相似度，节点=句子，边=相似度）
4. PageRank 迭代排序句子
5. 带多样性惩罚选择 Top-N 句（相似度 >0.8 的句子不重复选）

### TF-IDF 句子评分
1. 分割句子 + 分词
2. 计算每个词的 TF-IDF 权重
3. 句子评分 = 词 TF-IDF 之和 × 位置奖励（前20%句子×1.3） × 长度惩罚
4. Jaccard 相似度去重（>0.6 跳过）

## 快速使用
```python
from text_summarization import TextSummarizer

# TextRank
ts = TextSummarizer(method="textrank")
summary = ts.summarize("长文本...", num_sentences=3)
# → ["句子1", "句子2", "句子3"]

# TF-IDF
ts = TextSummarizer(method="tfidf")
summary_text = ts.summarize_text("长文本...", num_sentences=3)
# → "句子1。句子2。句子3。"
```

|------|--------------|
| 400余名老人遭养生诈骗... | "其中一起是400余名老年人遭健康养生诈骗" + "共计400余名老年人被骗，涉案金额3000万余元" |

## 算法对比
| 维度 | TextRank | TF-IDF |
|------|----------|--------|
| 速度 | 中等（需建图+迭代） | 快（线性扫描） |
| 语料依赖 | 无（单篇即可） | 无（单篇即可） |
| 多样性 | 显式控制（阈值） | Jaccard 去重 |
| 位置偏好 | 无（纯图结构） | 前20%句子加分 |
| 长文本 | 好（图结构稳定） | 好（密度计算稳定） |
