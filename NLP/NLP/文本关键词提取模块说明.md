# NLP 关键词提取模块说明

## 模块位置
- 核心代码：`keyword_extraction.py`
- 依赖预处理：`preprocessing.py`

## 概述
实现两种经典传统关键词提取算法：

| 方法 | 思路 | 需语料 | 适用场景 |
|------|------|--------|----------|
| **TF-IDF** | 词频×逆文档频率 | 是 | 有语料的大规模提取 |
| **TextRank** | 图排序(类PageRank) | 否 | 单篇即用 |

## 算法原理

### TF-IDF
TF = word_count / total_words, IDF = log((N+1)/(df+1)) + 1, Score = TF × IDF
*完全从数学公式实现，不依赖 sklearn*

### TextRank
1. 分词去停用词 → 2. 共现建图 → 3. PageRank 迭代 → 4. Top-N 输出
*从零实现加权邻接矩阵，处理悬挂节点*

## 快速使用
```python
from keyword_extraction import KeywordExtractor
ext = KeywordExtractor(method='tfidf')  # 或 'textrank'
ext.fit(corpus)  # TextRank 无需 fit
kw = ext.extract(text, top_n=10, with_scores=True)
```
