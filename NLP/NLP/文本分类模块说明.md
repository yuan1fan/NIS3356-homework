# NLP 文本分类模块说明

## 模块位置
- 核心代码：text_classification.py
- 依赖：preprocessing.py

## 概述
从零实现两种传统文本分类算法：

| 算法 | 原理 | 特点 |
|------|------|------|
| **朴素贝叶斯** | 词频先验+条件概率，Laplace平滑 | 简单高效，小样本表现好 |
| **逻辑回归** | 梯度下降，Sigmoid输出，L2正则 | 可解释概率，适合多分类 |

特征：词频 (Count) / TF-IDF（L2归一化）

## 快速使用
```python
from text_classification import TextClassifier
clf = TextClassifier(method="naive_bayes")  # 或 logistic_regression
clf.fit(texts, labels)
pred = clf.predict("新文本")
proba = clf.predict_proba("新文本")
report = clf.evaluate(test_texts, test_labels)
cv = clf.cross_validate(texts, labels, folds=5)
```

## 算法原理
朴素贝叶斯：P(c|doc) ∝ P(c) × ∏P(w|c)，对数计算避下溢
逻辑回归(OVR)：h(x)=sigmoid(w·x)，批量梯度下降+L2正则
